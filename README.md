# Traderson Calculator 📊

A professional trading calculator built as a Progressive Web App (PWA).
Works offline and can be installed on any phone or tablet.

## Features
- ✅ Basic & Scientific Calculator
- ✅ Pip Calculator (Forex, Gold, Indices, Crypto)
- ✅ Position Size Calculator (MyFXBook formula)
- ✅ Stop Loss — by Pips or exact Price
- ✅ Take Profit — by Pips or exact Price
- ✅ Profit & R:R Ratio calculation
- ✅ Margin Calculator
- ✅ Currency Converter (USD, EUR, GBP, JPY, HTG...)
- ✅ Works 100% offline (PWA)
- ✅ Installable on Android & iPhone

## Supported Pairs
Majors, Minors, JPY pairs, Gold, Silver, Oil, NAS100, US30, SPX500, DAX, BTC, ETH

## Install on Phone
1. Open the GitHub Pages URL in Chrome (Android) or Safari (iPhone)
2. Android: tap ⋮ → "Add to Home Screen"
3. iPhone: tap Share → "Add to Home Screen"

## Built with
HTML · CSS · Vanilla JavaScript · PWA (Service Worker)
